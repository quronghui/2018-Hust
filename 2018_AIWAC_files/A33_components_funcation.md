### (5) 电解电容器检测

## 三. Inductance
### (1) 分类
- 高频、中频、低频  
- 振荡、滤波、阻流、耦合
- 固定和可调
### (2) 主要参数
- 电感量、允许偏差、品质因数、分部电容及额定电流  
+ 电感量 -- 自感系数，自感应能力的物理量  
+ 单位：1H=1000mH=1000uH
- 品质因素：Q值，在某一频率的交流电压下，呈现感抗与其等效损耗电阻之比。Q越大，损耗越小，效率越高。
+ 分布电容：线圈的匝与匝之间，线圈与磁芯之间的电容，越小，稳定性越高。
### (3) 电感的检测
- 万用表：调至欧姆档接电感器两端  
- 电感器的 -- **直流电阻值**：与绕制线圈有直接关系  
+ R = 0; 内部短路性故障
+ R > 0; 电感器正常
### （4）电感线圈作用
+ 阻隔交流信号、滤波或组成谐振电路

## 四. 耦合和滤波
### (1) 耦合
- **耦合** 在电路中，是指一种电物理量，在相互隔离的情形中，由一方（电极或线圈）传给另一方的情形叫耦合。传递的是交流成分。 
- **耦合作用** 将本级信号传输到下一级电路中去，这种传输就叫耦合  
+ **电容耦合** 两个相互隔离电极，由于频率变化，电能从一个电极感应到另一个电极的过程
+ **电容耦合作用** 音频信号的 前级和功放之间通过电容连接，利用电容“通交流，隔直流”的特性，有直流电的传输，互不影响工作状态，从而传输有用信号。  
* **电感耦合** 两个相互隔离的线圈，由于电流的变化,  ?  
### (2) 区别
| 分类 | 耦合 | 滤波 |  
| :---: | :---: | :---: |
| 作用 | 传递有用的交流成分 | 过滤有害的交流成分 |
| 阻值大小 | C < 10uF | C > 100uF |
| 耐压大小 | 较低 | 较高 |

## 五. 二极管
### (1) 特性
- 单向导电性
- 最高反向电压：失去单向导电能力时的电压值
- 反向电流：在最高反向电压下，流过二极管的电流，反向电流低，单向导电性越好
### (2) 分类
- 检波二极管、整流二极管、变容二极管 、稳压二极管 、发光二极管 
### (3) 二极管的检测--电压档
+ 万用表：通电情况下，直流电压档？
+ 二极管正向压降：0.5V - 0.7V
+ 肖基特二极管压降：0.2V
+ 正向电流：I/mA
- 调换表笔：反向不导通，显示 “1”，正常，反之，损坏
- 若正、反测均为 0 或者 1 ，损坏
+ 反向电流:I/uA
### （4）二极管的检测--专用挡位--初步判断
+ 导通电阻--> PN节时 
+ 未上电：正向导通电流为1毫安、导通电压为0．75V，此时的电阻为0．75V／0．001A＝750欧姆
+ 上电：导通电流为1安、导通电压仍为0．75V（导通电压增加不多，可忽略）时，正向电阻为0．75V／1A＝0．75欧姆
- 反向电阻--> PN
+ 0L -- Over load
### （5） 二极管结构分类
+ PN结 —— P为正极（阳），N为负极（阴）

- **整流二极管**：将交流电变成直流电。
- **肖基特二极管**：工作于低电压、大电流的条件；导通压降0.4V，整流电流达到几千安
- **稳压二极管**：工作于反向击穿区；万用表正向接法——红-负，黑-正；正向R=10K,反向=无穷

## 六. 三极管
### (1) 定义

### (2) package

### (3) 特性
- 以基极电流微小的变化量，控制集电极电流较大的变化量
- β＝ΔIc/ΔIb 称为晶体三极管的电流放大倍数
- 万用表：hFE挡 Hybrid Forward commno emiter 可以测量
### (4) 工作状态
| 状态 | 要求 | 状态 | 作用 |
| :--: | :--: | :--: | :--: |
| 截止状态 | V(发) < V(PN) | I (基) = 0，I(集)=I(发)=0 | 失去电流放大作用，集电极和发射极开关断开状态|

## 八. 熔断器--保险丝
### (1) Feature
- 短路保护、严重过载保护
+ **熔体额定电流** 在规定工作条件下，长时间通过熔体而熔体不熔断的最大电流值。
+ **熔断器额定电流** 保证熔断器长期正常工作的电流。
- 一个额定电流等级的熔断器可以配用若干个额定电流等级的熔体
- 熔体的额定电流 < 熔断器的额定电流
### (2) Application
- 在一定时间内熔断
+ 负载超过某值：要求保护熔体在一定时间内熔断
+ 短路电流作用下和超过允许过负荷电流时
- 不用熔断
+ 过载能力，设备允许在一定条件下较长时间运行
+ 设备起动电流很大，但起动时间很短
### (3) 熔断器的检测
+ 万用表：欧姆档
+ 熔断器是好的 --> R < 5 欧；
+ 熔断器是坏的 --> R = 00.0

## 九、电声器件的识别与检测
### （1）define
+ 电信号 <---> 声信号 的相互转换。震动产生声音
+ 扬声器接收的是模拟信号吗？频率——决定模拟信号吗？
### (2)**阻抗匹配** 是啥呢？
- 阻抗--电阻、电容抗及电感抗在向量上的和
- 频率愈高则容抗愈小感抗愈大；电容--阻直流，电感--阻交流
- 低频电路中，我们一般不考虑传输线的匹配问题，只考虑信号源跟负载之间的情况，因为低频信号的波长相对于传输线来说很长，传输线可以看成是“短线”
- 结论：如果我们需要输出电流大，则选择小的负载R；如果我们需要输出电压大，则选择大的负载R；如果我们需要输出功率最大，则选择跟信号源内阻匹配的电阻R
- P = =U2/{[(R-r)2/R]+4×r}
+ 信号放大源阻抗匹配良好：后一级输入阻抗 》 5-10倍 （前一级的输出阻抗）
+ 输入端阻抗匹配时：传输线获得最大功率；
+ 输出端阻抗匹配的情况下：传输线上只有向终端行进的电压波和电流波，也就是功率，携带的能量全部为负载所吸收
- 阻抗匹配实现： R 负 / R 信号源 =  K(1) 共轭现象出现，实现功率最大的传输
- 直流电源来讲，阻抗匹配时输出效率只有50%
### (3) Chose
+ 灵敏度高、失真小、频响宽
+ 阻抗匹配
+ 增加保真度：功放级输出功率 》 2-3倍 扬声器额定功率 

## 十、迟滞比较器的使用
### 1、无反馈的迟滞比较器特点
+ U(i) 输入电压<----->  U(R) 参考电压
- U（i）同相输入 : U(i) >  U(R) ----> U (om) 为正
- U（i）反相输入 : U(i) <  U(R) ----> U (om) 为正
### 2、在正反馈条件下，正向有输入电压U（R）
#### (1) 为什么 参考电压U(H) > U(R) ?
+ 看电路图中的参靠电压 U(H)
+ U(H) = U(om) * [ R1 / (R1 + R2) ] + U(R) *[ R2 / (R1+R2) ] 
+ 约等于 U(R) + U(om)
+ U(H) > U(R)，在右边
#### （2）V(om)正负确定
+ 反向输入确定：U(i) >  U(R) ----> U (om) 为正
+ U (om) 是一个定值，只是有两种状态
+ 一开始的参靠电压一定是 U(R) = U(H),因为提供了外部电源的
### 3、我们用的Touch状态计算
+ U(i) 输入电压 ----> 提供了外部电源 VCC; U(R) 参考电压也是外部电源
- 结论 ：U(i)触发的条件为 V(L) ，当 U(i) 下降至V(L)才会触发
- 迟滞特性： U(i),增加至 V(H) ,才会发生跳变
